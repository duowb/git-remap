# git-remap

本地批量更新git源地址


### 使用方法 1 

> 本地部署方案

1. 安装 [`nodejs`](https://nodejs.org/),最好是`v20`以上版本

2. 运行 `npx git-remap view`

3. 输入 项目路径,会递归向下查找该路径下的所有git项目


### 使用方法 2
> 纯前端方案
